*Last updated: 2026-05-05*

다음 세션의 Claude는 포트폴리오 자산 구성 작업 전에 이 문서를 읽으세요.

## 핵심 인사이트 (실증 근거)

**분석 기간**: 2021-12-01 ~ 2026-04-24 (4.4년), 6자산 51개 조합

| 구성 타입 | 대표 조합 | 누적수익 | MDD | Sharpe |
|-----------|-----------|---------|-----|--------|
| 히든카드 only | RKLB+VRT | +1342% | -51% | 1.43 |
| 효율적 only | GLD+BRKB+QQQ+ITA | +60% | -24% | 0.98 |
| **혼합 (균형)** | **GLD+ITA+VRT** | **+351%** | **-16%** | **1.76** |
| 혼합 (공격) | GLD+RKLB+VRT | +771% | -40% | 1.69 |

**핵심 발견:**
1. **히든카드 단독**: 수익 최대지만 MDD가 감당 불가 수준
2. **효율적 자산 단독**: 전략이 B&H도 못 이김 (모델 알파 없음)
3. **혼합**: 효율적 자산이 MDD를 줄이고, 히든카드가 수익 엔진 역할

## Layer 0 알고리즘

### 포트폴리오 구성 규칙

```
포트폴리오 = [앵커 자산] + [섹터 ETF] + [히든카드]
```

#### 1. 앵커 자산 (필수, 1개)
- **역할**: MDD 완충, 시장 하락 시 방어
- **기준**: 낮은 주식 상관관계, 안전자산, 위기 시 상승
- **현재**: GLD (금)
- **대체 후보**: TLT (국채), SLV (은), 현금 ETF

#### 2. 섹터 ETF (선택, 0~2개)
- **역할**: 트렌드 방향 포착, 분산
- **기준**: 광범위한 섹터 ETF (개별 종목 아님), 5년+ 역사
- **현재**: ITA (방산), QQQ (기술)
- **대체 후보**: XLE (에너지), XLK (기술), XLF (금융), XBI (바이오)
- **선택 기준**: 현재 매크로 트렌드가 지지하는 섹터

#### 3. 히든카드 (선택, 1~2개)
- **역할**: 비대칭 수익 포착 (하나 터지면 포트폴리오 전체 커버)
- **기준**: 신흥 성장 섹터의 개별 기업, 높은 변동성 허용
- **현재**: RKLB (우주), VRT (데이터센터)
- **선택 시점**: 섹터가 어느 정도 가시화된 후 (완전 초기는 아님)
- **주의**: 미리 예측 불가능 → 나중에 추려지면 편입

## 비율 가이드라인

### MDD 우선 (안전 중시)
```
앵커 : 섹터ETF : 히든카드 = 2 : 1 : 1  (예: GLD×2, ITA, VRT)
또는                          1 : 1 : 1  (예: GLD, ITA, VRT)
```
- 예상 MDD: -15 ~ -20%
- 예상 CAGR: +25 ~ +40%
- 대표 결과: **GLD+ITA+VRT** → 4.51x, MDD -15.8%, Sharpe 1.76

### 수익 우선 (공격 중시)
```
앵커 : 섹터ETF : 히든카드 = 1 : 1 : 2  (예: GLD, QQQ, RKLB+VRT)
또는                          1 : 0 : 2  (예: GLD, RKLB+VRT)
```
- 예상 MDD: -35 ~ -50%
- 예상 CAGR: +50 ~ +80%
- 대표 결과: **GLD+RKLB+VRT** → 8.71x, MDD -40%, Sharpe 1.69

### 현재 운영 (균형)
```
앵커 : 섹터ETF : 히든카드 = 1 : 1 : 1 ~ 1 : 2 : 1
```
- 라이브: GLD, BRK-B, QQQ, RKLB (히든카드 비율 낮음)
- 개선안: GLD, ITA, VRT (또는 GLD, QQQ, ITA, VRT)

## 자산 교체 판단 기준

### 히든카드 편입 조건
1. 섹터 성장 스토리가 언론/기관에서 언급되기 시작 (완전 초기는 아님)
2. 매출/기술 진척이 확인된 개별 기업 존재
3. 주가 변동성이 높음 (스탠다드 ETF보다 베타 > 1.5)
4. 앵커/섹터ETF와 낮은 상관관계

### 히든카드 제거 조건
1. 섹터가 성숙해서 ETF화됨 (더 이상 "히든"이 아님)
2. 회사 펀더멘털 악화 (매출 급감, 경쟁 심화)
3. MDD가 -70% 초과 시 검토

### 효율적 자산 교체 조건
1. 매크로 트렌드 변화 (예: 금리 상승 → 국채 불리)
2. 섹터 ETF 내 모멘텀 사라짐 (2년 이상 하락 추세)

## 왜 Layer 0가 필요한가

```
Layer 0 (Human):  어떤 자산군? 비율은? ← 이 문서
    ↓
Layer 1 (Algo):   80개 전략 → 피처 행렬
    ↓
Layer 2 (ML):     언제, 얼마나 투자?
    ↓
Layer 3 (SINDy):  비선형 항 확장 (미래)
```

Layer 0이 잘못되면 Layer 1~2가 아무리 좋아도 포트폴리오 전체가 나빠짐.
반대로 Layer 0에서 히든카드를 잘 선택하면 Layer 1~2의 알파가 극대화됨.

**Layer 0의 역할**: "기계가 잘 작동할 운동장을 만들어주는 것"
- 효율적 시장 자산 → 모델 알파 없음 (운동장이 너무 평평)
- 히든카드 → 모델 알파 극대 (운동장에 기회가 많음)
- 혼합 → MDD 방어막 + 알파 포착
