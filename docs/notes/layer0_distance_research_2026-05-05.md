## 연구 배경

사용자가 비레이블 학습(unsupervised learning)을 공부하며 아이디어 제안:

> "모든 자산군을 비레이블 학습으로 학습시켜, 수학적으로 정의한 거리가 먼 자산끼리
> 조합하면 앵커 효과가 나서 MDD가 낮아질 것이다."

이것이 **Layer 0** — 어떤 자산을 선택할지를 사람의 판단이 아닌 수학이 결정.

---

## 실험 설정

- **자산**: GLD, BRK-B, QQQ, RKLB, ITA, VRT (6개)
- **기간**: 2021-12-01 ~ 2026-04-24 (846 거래일, 4.4년)
- **조합**: 2자산(15) + 3자산(20) + 4자산(15) + 6자산(1) = **51개**
- **검증**: 각 조합의 평균 거리 vs 실제 MDD (Pearson/Spearman 상관)
- **스크립트**: `scripts/layer0_distance_analysis.py`
- **결과 파일**: `outputs/layer0_distance_analysis/`

## 5가지 거리 메트릭 결과

| 순위 | 방법 | 개념 | B&H MDD r | 전략 MDD r | 유의성 |
|------|------|------|-----------|-----------|--------|
| 🥇 | **DTW** | 시계열 모양 유사도 | **-0.873** | **-0.716** | p<0.001 |
| 🥈 | **Wasserstein** | 수익률 분포 거리 | -0.727 | -0.547 | p<0.001 |
| 3위 | Tail Dependence | 폭락 동조도 | +0.304 | +0.230 | p<0.05 (역방향) |

**r < 0 = 거리 멀수록 MDD 낮음 = 가설 지지**

## 핵심 발견

### 1. 가설 부분 입증
DTW(r=-0.87)와 Wasserstein(r=-0.73)에서 **통계적으로 유의하게** 가설 지지.
"시계열 모양이 다를수록, 수익률 분포가 다를수록 MDD가 낮아진다."

### 2. 상관계수는 쓸모없음
r=+0.23 — 기존 금융의 상관계수 기반 분산투자가 왜 위기 때 실패하는지 수치로 확인.

### 3. PCA와 Tail Dependence 실패 이유
GLD가 모든 자산과 너무 다른 outlier라서 두 방법 모두 GLD에 의해 왜곡됨.
→ 해결: 자산을 클러스터링 후 클러스터 내 거리를 측정, 또는 GLD 제외 후 재분석.

### 4. DTW가 최강인 이유
- 평균적 관계(상관계수) 아니라 **시간축 이동을 허용한 모양 비교**
- 위기 시 동조 패턴이 다른지를 잘 포착
- 비선형, 시간 불변 → 위상적(topological) 접근과 가장 유사

## 다음 연구 방향 (시험 후)

### 단기 (바로 할 것)
1. **DTW 거리로 자산 선택 알고리즘 구현**
   - N개 자산 입력 → DTW 거리 행렬 → 가장 먼 조합 자동 선택
   - "최적 k자산 조합 찾기" = 평균 pairwise DTW 최대화

2. **더 많은 자산으로 확장**
   - 현재: 6자산만 테스트
   - 목표: 수백 개 자산 (커머디티, 채권, FX, 암호화폐 포함)
   - DTW 계산: N×N 행렬, N=100이면 4950쌍 → 병렬화 필요

3. **GLD outlier 문제 해결**
   - 자산을 클러스터링 먼저 → 각 클러스터에서 대표 자산 선택
   - UMAP/t-SNE로 시각화

### 중기 (비레이블 학습 심화)
4. **Autoencoder 잠재 공간 거리**
   - PCA 실패 이유: 선형 변환 → GLD outlier 문제
   - VAE(Variational Autoencoder): 비선형 → outlier 처리 가능
   - 각 자산 = 잠재 벡터 → 코사인 유사도 or 유클리드 거리

5. **Topological Data Analysis (TDA)**
   - 사용자가 공부 중인 위상적 접근 직접 적용
   - Persistent Homology: 수익률 점군의 "구멍" 구조 비교
   - giotto-tda 라이브러리

6. **시간 변동 거리 (Time-varying distance)**
   - 거리는 시간에 따라 변함 (평상시 vs 위기)
   - Rolling DTW: 6개월 윈도우로 거리가 어떻게 변하는지
   - 위기 전조 신호로 활용 가능?

### 장기 (Layer 0 완성)
7. **Layer 0 완전 자동화**
   ```
   입력: 수백 개 자산 시계열
       ↓ DTW/Autoencoder 거리 행렬
       ↓ 클러스터링 (앵커/섹터ETF/히든카드 자동 분류)
       ↓ 최적 k자산 조합 선택
   출력: active_universe.json 자동 생성
   ```

## 연구 의의

기존 금융의 자산 선택:
- Markowitz: 상관계수 행렬 최적화 (선형, 시간불변 가정)
- Risk Parity: 리스크 기여도 균등화 (여전히 공분산 기반)

이 접근의 차별점:
- **비선형 거리** (DTW, Wasserstein) → 위기 시 동조 패턴 포착
- **비레이블 학습** → 사람이 섹터 라벨 없이도 자산 구조 파악
- **위상적 관점** → 절대값이 아닌 패턴 구조로 자산 분류
- **실증 근거 있음** → DTW r=-0.87, p<0.001 (51개 조합, 4.4년)
