**핵심 발견**: 새로 추가된 랭크 구간이 항상 가장 큰 기여를 함.

Lasso/Ridge가 상관이 높은 기존 피처를 다운웨이팅하고 새 orthogonal 피처에 가중치를 줌.

### Sharpe peak: top-70 (280 컬럼) — 단, 차이 미미 (1.12~1.15)

| 기준 | 판단 |

## 7. 향후 과제

1. **h45 전용 모델** — `--target-horizon-days 45 --hold-days 45` 학습+투자 일치 실험
2. **Exposure cap 도입** — 최대 노출도 60% 제한 시 top-40~50 효율 변화
3. **오차 곡선 (MSE vs N)** — U자/double descent 확인
4. **BRK-B horizon scan** — 5일 간격 스캔으로 최적 horizon 발견 후 config 업데이트
5. **타 자산 horizon scan** — SPY/QQQ/TLT/RKLB 각각 최적 horizon 찾기
6. **각 자산 top-N 실험** — BRK-B/SPY에서도 peak N 확인
