## 세 단계 접근법 개요

| 단계 | 이름 | 방법 |
|---|---|---|
| A | **Single Signal** | 4개 전략 중 best rank1 단독 운용 |
| B | **Signal Ensemble** | 4개 전략 rank1씩 → 앵커마다 long-only weight 재최적화 조합 |

이 리포트는 B vs C 비교. 동일 조건: 1M 업데이트, 2020~2024, h130/h45.

## 핵심 관찰

### 1. h130에서 ML-Guided 압도적 우위
- 수익: 모든 연도에서 C > B
- 특히 2024: B(+8.0%) vs C(+16.4%) — 약 2배 차이
- 2022 방어: B(-2.3%) vs C(-0.2%) — C가 하락도 더 잘 방어

### 2. Baseline B의 노출도 구조적으로 낮음
- B h130 노출도: 23~42% 수준 (Fear-Greed 제외 후에도 유사)
- C h130 노출도: 29~65% 수준
- B는 조합 신호가 약해 포지션을 키우지 못함
- C는 ML 예측이 확실할 때 노출도를 높임 → 강세장에서 자본 활용 우수

### 3. h45에서 재계산 후 C 전 구간 우위 (수익 기준)
- 이전 버전: h45 2021에서 B 우위(-3.4% vs +0.0%)
- 재계산 후: h45 2021에서 C=+1.5% vs B=+0.0% → C 우위로 전환
- Fear-Greed 버그가 ML 모델을 교란하던 효과 제거됨

### 4. 슬라이드 스토리 요약
```
Signal Ensemble (B): 4전략 조합 → 낮은 노출도, 낮은 수익
ML-Guided (C):       모델 선택  → 적응적 노출도, 높은 수익
→ ML 모델 선택의 핵심 가치 = 언제 얼마나 베팅할지를 학습
```
