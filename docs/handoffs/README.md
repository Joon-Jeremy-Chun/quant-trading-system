## 규칙

- 새 세션 handoff는 이 폴더에 `session_handoff_YYYY-MM-DD_final.md` 형식으로 저장
- `_final`: 최종 숫자 및 완료 상태 포함한 확정본
- `_notes`: 메모/초안 성격 버전 (선택적)
- 원본 파일은 `docs/notes/`, `docs/reports/`에 그대로 유지
